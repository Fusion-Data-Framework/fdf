factory.py module¶

factory.py - root module for the FDF package.

Classes¶

Machine - root class for the FDF package Shot - shot container class Logbook - logbook connection class Container - diagnostic container class Node - mdsplus signal node class FdfError - error class for FDF package

Usage¶

>>> import fdf
>>> nstx = fdf.Machine('nstx')
>>> nstx.s140000.logbook()
>>> nstx.addshots(xp=1048)
>>> nstx.s140000.mpts.plot()
